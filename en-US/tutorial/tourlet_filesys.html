Feature Showcase: Store a Filesystem Layout in a Project


You can import a filesystem structure into a static project (or within a folder in a project or the toolbox) using the Import from Filesystem function. Folders will be created for each directory, containing the same files as on the filesystem. This is a one-time import, to create a project or folder that stays in sync with the filesystem, see the Live Projects feature showcase.

Right-click a folder in a project and select Import from File System.




Select the directory, specify inclusion and exclusion parameters, and select Import directory structure as the folder structure.




The imported file system has the same structure as the disk.
