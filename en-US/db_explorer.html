Database Explorer Komodo IDE only

The Database Explorer is a tool for examining the structure and content of databases. By default, it support SQLite and Oracle, but support for addional database types is avaialable through extensions.

Databases Sidebar

Note: The Database Explorer is still under active development. The interface will undergo extensive changes during the Beta period.

Open the Databases sidebar from the List all Sidebars menu button in the left pane. Use the Add a Database Connection button to configure a connection. Select a Database Type and configure the required connection information or the path to the file.

Click Add to create the connection and open the database view in the sidebar.

Each database appears as an top-level node in the sidebar. Clicking on the arrow next to the database expands a tree showing all tables. Table names also expand to show the contained field names.

Examining Tables

Double-click on a table name in the sidebar to open the Database Explorer dialog. The dialog box displays:

  • Full Table: a spreadsheet-like view of the selected table, showing all fields.
  • Query: a Query field for composing SQL queries (for any table) and an Output pane to show the query results.
  • Schema: displays the Schema for the selected table including the field id, name and type as well as whether it is nullable. Indicates the default value (if any) and shows which is the primary key.
