Welcome to the Komodo User Guide

What's new in Komodo Edit

  • Syntax Checking: Komodo's syntax checking feature has been greatly improved in Komodo 7:
    • Multi-language syntax checking - so for a HTML file you'll now receive errors and warnings on the HTML, JavaScript and CSS content.
    • A syntax checking tab has been added to the bottom pane - so you can easily see the list of errors and warnings in the one place.
    • There is now a wider ranger of syntax checkers to choose from, such as the addition of JSLint for JavaScript and PyLint for Python.
    • Enable/disable syntax checkers on a per-file or per-language level. Komodo's syntax checking preferences now offer complete customization of syntax checking parameters.
  • Updated Mozilla Code Base: Komodo is now based on the Mozilla Firefox 4 code base. This update brings a number of benefits for Komodo developers:
    • Faster startup - Komodo 7 does not have any component registration, so the first startup and restarting after installing an add-on will be substantially faster.
    • A better browser preview - supporting the lastest and greatest web technologies.
    • More stable - the stability improvements to the underlying Mozilla code base will make Komodo itself more stable.
  • New Languages
    • Mojolicious - a Perl templating language
    • CoffeeScript - a JavaScript templating language
