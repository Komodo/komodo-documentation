  • What's new in Komodo IDE

    • Sections List: A new menulist in the status bar which lets you see, and quickly jump to, code objects in the current file.
    • SCC Changelists and Multi-File Commit: A new tab in the right pane for managing changelists. An updated SCC commit dialog allows you to commit multiple files in a single SCC operation and display multi-file diffs.
    • Unit Testing Integration Improvements: Unit test harnesses can now be added to Komodo as extensions. Test plans can be saved in an individual file's preferences. Global test plans are deprecated, but existing ones can be migrated into file preferences.

    (Read the full Release Notes.)
