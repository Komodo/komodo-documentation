  • What's new in Komodo Edit

    • Built on Mozilla 1.9: Komodo is now built on the Mozilla 1.9 branch, the same code base Firefox 3.0 is built on. This includes a number of bug fixes and performance improvements.
    • Multi-Window: Komodo can now open multiple top-level windows, allowing you to configure your workspace the way you want when editing or debugging.
    • Numerous bug fixes and other minor improvements.
