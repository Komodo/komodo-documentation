  • What's new in Komodo IDE

    • Unit Test Integration allows you to run your test suites within Komodo and find failure points in your source code fast.
    • Find/Replace improvements: A better interface for searching and replacing, plus new Find in Project and Replace in Files features.
    • Abbreviations: Quickly insert code snippets by entering abbreviations for them. Several useful default snippets are included, and additional ones can be added easily in the Toolbox.
    • Improved Source Code Control: Updates are now asynchronous, so commands work in the background while you keep working.
