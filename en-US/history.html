History

The History feature lets you navigate backwards and forwards through your editing sessions. Similar to the history feature provided by a web browser, Komodo provides a Go back... and Go forward... buttons, and a drop down list of recent editing positions.

Positions

The editing history is saved as a list of positions. Each position is made up of a filename and a cursor position. Positions are window-specific (as opposed to tab-specific) so each Komodo window will have it's own.

Positions are saved to the history by the following actions:

  • opening a new file
  • changing to another open file and making an edit
  • jumping to a line from the Find Results tab
  • using Go To Definition

History Navigation

You can navigate the history using the Go back... and Go forward... buttons, or by selecting the desired position from drop down list.

Forward and Back are also available in the Code menu, and will display the keybindings assigned to these actions ("Editor: Go Forward" and "Editor: Go Back" in the Key Binding preferences).
